## Install and Set Up

### Local (with Docker)

```
$ docker-compose build
$ docker-compose up  # Startup http://localhost:3030/
```

```
$ docker-compose down  # Shutdown
```

### IBM Cloud (stand-alone)

```
$ ibmcloud cf push
```
